# Tangyi Agent Activation Checklist

Complete this checklist after joining the workspace.

## Pre-Activation

- [ ] Clone/sync workspace from GitHub
- [ ] Read workspace AGENT-START-HERE.md
- [ ] Enter family-dinner project

## First Read

- [ ] Read CURRENT.md - understand today's state
- [ ] Read ROLES.md - understand your authority
- [ ] Read shared/rules.md - understand business rules
- [ ] Read shared/WRITE-POLICY.md - understand event mapping

## Understanding

- [ ] Know where CURRENT.md is and how to update it
- [ ] Know where dish-history.md is and how to record feedback
- [ ] Know where notifications/ is and when to use it
- [ ] Know where incoming/ is and when to use it
- [ ] Understand your write scope
- [ ] Understand what requires asking Tangyi

## Validation

- [ ] Run `sh scripts/check-project.sh`
- [ ] Fix any errors reported

## Report

- [ ] Record activation completion in CURRENT.md
- [ ] Mark task FD-001 as done in TASKS.md
- [ ] Notify Agent-A via notification or @handoff

## Blocker Report

If any step fails, record in CURRENT.md open items:
- What failed
- What you expected
- What you tried
- What would help
