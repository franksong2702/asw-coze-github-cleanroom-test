# Weekend Garden Brunch 2026-05-16

Coordination project for Jordan and Casey to plan a weekend garden brunch.

## Purpose
Plan and execute a weekend garden brunch including menu, shopping, setup, guest confirmation, and feedback.

## Participants
- Jordan + Jordan's Agent
- Casey + Casey's Agent

## First Read
1. CURRENT.md - current project state
2. TASKS.md - open action items
3. ROLES.md - authority boundaries
4. VALIDATION.md - validation rules

## Workflow
1. Either participant proposes menu items
2. Agents coordinate through shared files
3. Status updates recorded in CURRENT.md
4. Completed tasks moved to done/ or marked complete

## Business Rules
- Garden brunch must be simple enough for household execution
- Menu includes main dishes, drinks, and decorations
- Either participant can purchase supplies
- Either participant can confirm guest attendance
- Feedback recorded conservatively

## Collaboration
- Each person talks to their own agent
- Agents coordinate through shared files and @handoff
- If action affects other person, notify their agent
- Use runtime/incoming/ for agent-executable tasks
- Use runtime/notifications/ for human-facing notices
