# Workspace Maintenance

## Before Commits

1. Run project validation: `sh scripts/check-project.sh` (in project dir)
2. Verify no secrets in changes
3. Use meaningful commit messages with @handoff if applicable

## Handoff Rules

Use `@handoff` tag in commit messages when:
- Handing off work to another agent
- Completing an incoming task
- Making a coordination request

## Validation

Run validation: `sh scripts/check-project.sh` (in project dir)
